1. Что пишут на .NET Core?
- На .NET Core пишут приложения используя различные языки программирования такие как C#, F#, и VB.NET.

2. Что такое MSIL и какие задачи он выполняет в .NET?
- MSIL (Microsoft Intermediate Language) представляет собой промежуточный язык в который компилируется 
код на языке .NET. MSIL выполняет роль промежуточного представления которое обеспечивает переносимость 
кода и возможность его выполнения на различных платформах.

3. Какие языки программирования поддерживаются .NET Framework?
- .NET Framework поддерживает несколько языков программирования включая 
C#, Visual Basic .NET (VB.NET), F#, C++/CLI, и др.

4. Какую роль MSIL играет в обеспечении переносимости кода между различными платформами?
- CLR (Common Language Runtime) обеспечивает безопасность типов с помощью метаданных 
проверки типов во время компиляции и выполнения, а также управление памятью через сборку мусора.

5. Как CLR обеспечивает безопасность типов и управление памятью в приложениях .NET?
- Общеязыковая спецификация (CLS) определяет стандарты которым должны соответствовать 
языки программирования для обеспечения их взаимодействия в среде .NET. CLS 
используется для создания совместимого кода, который может быть использован на различных языках .NET.

6. Каковы основные принципы Общеязыковой спецификации (CLS) и что это такое
используется для?
- CTS (Common Type System) и CLS способствуют совместимости кода написанного на различных языках .NET. 

7. Как CTS и CLS способствуют совместимости кода написанного на разных платформах .NET. языки?
- CTS определяет типы данных и их поведение во всех языках .NET, а CLS уточняет правила 
которые языки должны соблюдать для обеспечения взаимодействия.

8. Что такое оптимизация на основе профиля (PGO) и как она может повысить производительность приложений .NET? 
Как можно использовать технологию SIMD для оптимизации вычислений в приложениях .NET?
- Оптимизация на основе профиля (PGO) в .NET позволяет улучшить производительность приложений
адаптируя оптимизации к конкретному профилю использования приложения. 
SIMD (Single Instruction, Multiple Data) — это технология позволяющая 
выполнять параллельные вычисления на векторах данных что может быть 
использовано для оптимизации вычислений в приложениях .NET.
