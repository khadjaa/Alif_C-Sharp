#39. Database. SQL. PostgreSQL. Practice №6

1. Приведите примеры команд DDL, таких как CREATE, ALTER, DROP, TRUNCATE, COMMENT и RENAME, 
и объясните их соответствующие функции.
- Команды DDL (Data Definition Language) включают CREATE (создание объектов, таких как 
таблицы), ALTER (изменение структуры объектов), DROP (удаление объектов), TRUNCATE (очистка 
данных без удаления самой таблицы), COMMENT (добавление комментариев к объектам) и RENAME 
(переименование объектов). Эти команды позволяют управлять структурой базы данных.

2. Объясните основные функции команд DML, включая SELECT, INSERT, UPDATE и DELETE, и то, 
как они способствуют управлению данными в реляционной базе данных.
- Команды DML (Data Manipulation Language), такие как SELECT (извлечение данных), INSERT 
(вставка новых данных), UPDATE (изменение существующих данных) и DELETE (удаление данных), 
используются для управления содержимым таблиц и обеспечения целостности данных.

3.Выделите особенности, 
отличающие PostgreSQL, такие как поддержка расширенных типов данных и процедурных языков. 
Также упомяните примечательные особенности MySQL и SQL Server.
- PostgreSQL отличается поддержкой расширенных типов данных и процедурных языков, таких 
как PL/pgSQL. MySQL и SQL Server также имеют свои уникальные особенности, такие как 
широкая популярность MySQL в веб-разработке и масштабируемость SQL Server для крупных 
предприятий.

4. Объясните концепцию движков хранения данных в MySQL и то, как они влияют на поддержку 
транзакций и производительность.
- В MySQL концепция движков хранения данных определяет способ хранения и обработки данных, 
что влияет на поддержку транзакций и производительность. Например, InnoDB обеспечивает 
поддержку транзакций и ACID-совместимость, а MyISAM обычно используется для приложений 
с высокой производительностью чтения.

5. Обсудите поддержку типов данных JSON и JSONB в PostgreSQL и их значение. Как SQL Server 
расширил свою совместимость с различными платформами и каковы его исторические ассоциации 
с точки зрения операционных систем?
- В PostgreSQL поддерживаются типы данных JSON и JSONB для работы с JSON-структурированными 
данными, что позволяет хранить, извлекать и манипулировать JSON-объектами в базе данных. 
SQL Server расширил свою совместимость с различными платформами, а его исторические 
ассоциации связаны с операционными системами Microsoft.

6. Объясните основные компоненты таблицы, такие как столбцы и строки. Какую информацию 
представляет столбец и из чего состоит строка?
- Основные компоненты таблицы включают столбцы (определяют типы данных) и строки 
(содержат фактические данные). Столбец представляет определенный тип данных, такой 
как числа или строки, в то время как строка содержит фактические значения для каждого 
столбца.

7. Дайте определение транзакции в контексте реляционной базы данных и объясните ее 
важность для поддержания согласованности и целостности.
- Транзакция в реляционной базе данных - это логическая операция, состоящая из одного или 
нескольких SQL-запросов, которая должна быть выполнена атомарно (полностью или не вообще)
и согласованно (сохранение целостности данных). Важность транзакций заключается в 
обеспечении согласованности и целостности данных в базе данных при выполнении множества 
операций.
