1.  Какова роль главной точки входа в приложении .NET?
- Роль главной точки входа (Main) в приложении .NET - Основная точка входа где начинается выполнение приложения.
Место где CLR (Common Language Runtime) начинает выполнение кода. Обычно находится внутри класса 
и содержит метод Main который вызывается при запуске приложения.

2.  Как можно запустить .NET-приложение с помощью команды dotnet run?
- Можно запустить .NET-приложение с помощью терминала
и ввести команду dotnet run и нажмите Enter. Это автоматически компилирует и запускает приложение.

3.  Каково назначение Pack Application в .NET, и какой код позволяет его использовать?
- dotnet pack - упаковывает приложение в NuGet-пакет готовый для распространения.
Пример использования:
dotnet pack -c Release

4.  Какие типы проектов можно создавать в .NET и для чего они нужны?

Console Application (Консольное приложение)
Создает приложение для выполнения в консоли.

- dotnet new console -n MyConsoleApp

Class Library (Библиотека классов)
Создает библиотеку классов для использования в других проектах.

- dotnet new classlib -n MyLibrary

ASP.NET Core Web Application (Веб-приложение ASP.NET Core)
Создает веб-приложение на основе ASP.NET Core.

- dotnet new web -n MyWebApp

Unit Test Project (Проект для юнит-тестов)
Создает проект для написания и запуска юнит-тестов.

- dotnet new xunit -n MyTests
