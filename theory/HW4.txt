1. Почему типы значений хранят данные напрямую, а ссылочные типы хранят ссылки? 
- Типы значений хранят данные напрямую потому что они представляют собой 
самостоятельные объекты которые содержат свои данные внутри себя. 
Ссылочные типы хранят ссылки потому что они представляют собой 
указатели на объекты которые фактически хранятся в другом месте в памяти.

2. Какие типы данных обычно являются типами значений, а какие - ссылочными типами в C#?
- Типы значений включают в себя простые типы данных такие как int float
bool char а также структуры и перечисления. Ссылочные типы включают 
классы интерфейсы делегаты строки и массивы

3. Как объявить переменную с типом Nullable? Каково ее значение по умолчанию? 
- Для объявления переменной с типом Nullable в C# используется суффикс "?"

int? nullableInt = null;

Значение по умолчанию для типа Nullable равно null

4. Как проверить, имеет ли переменная Nullable значение null? 
- Для проверки можно использовать оператор условия или метод HasValue
if (nullableInt == null)
{
    
}

if (!nullableInt.HasValue)
{
    
}

5. В каких ситуациях может быть полезен оператор ??= (оператор нулевой ссылки)? Приведите примеры.
- Оператор ??= (оператор нулевой ссылки) используется для присвоения переменной 
значения по умолчанию только если она имеет значение null

int? value = null;
value ??= 42;

6. Что означает оператор ??? (оператор коалиции нулей)? Как он помогает обрабатывать нулевые значения? 
- Оператор ??? (оператор коалиции нулей) используется для возврата первого ненулевого 
операнда из списка. Если все операнды равны null возвращается null.

int? value1 = null;
int? value2 = 42;
int result = value1 ??? value2;

7. Какие данные обычно хранятся в памяти HEAP и как ими управляют в .NET?
- В памяти HEAP хранятся динамически выделяемые объекты такие как экземпляры классов. 
Управление памятью в .NET для объектов в HEAP осуществляется автоматически сборщиком мусора.

8. Какие данные хранятся в памяти STACK и чем они отличаются от данных в HEAP? 
- В памяти STACK хранятся локальные переменные и данные метода а также 
информация о вызовах методов. Данные в STACK имеют ограниченный 
срок жизни и освобождаются после завершения метода.

9. Где в .NET размещаются переменные типов значений и ссылочных типов? 
Какие переменные выделяются в STACK, а какие - в HEAP?
- Переменные типов значений как правило размещаются в STACK если они 
являются локальными переменными метода. Переменные ссылочных типов могут 
храниться в HEAP а ссылки на них в STACK.
