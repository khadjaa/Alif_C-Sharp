1. Какие два основных способа создания программы на языке C# вы изучили на уроке?
- Console Application (Консольное приложение) Создает приложение для выполнения в консоли
команда: dotnet new console -n MyConsoleApp
- Class Library (Библиотека классов) Создает библиотеку классов для использования в других проектах
команда: dotnet new classlib -n MyLibrary

2. Чем операторы верхнего уровня отличаются от традиционного метода Main?
- Операторы верхнего уровня (Top-level statements) это новая функциональность в C# 9.0
позволяющая писать код без явного объявления метода Main. Код находится прямо на верхнем уровне файла
Традиционный метод Main это стандартный метод с которого начинается выполнение программы
Он объявляется внутри класса

5. Что такое псевдонимы типов и как они могут быть полезны в коде?
- Псевдонимы типов это короткие имена для существующих типов данных 
Например using Age = System.Int32; позволяет использовать Age вместо System.Int32 в коде

6. Какие операции можно выполнять над переменными строкового типа в C#?
- Конкатенация: Объединение строк с использованием оператора +
  Индексация: Обращение к отдельным символам строки

8. Какие методы работы со строками вы изучили в классе? Опишите, для чего они могут быть использованы.
- Посимвольное создание строки с помощью оператора new char, создание строки с помощью типа данных string, 
и разница между ними в том, что в памяти строка созданная с помощью new char хранит 
каждый символ в отдельной ячейке памяти, а string хранится в одной ячейке памяти. 
Так же при создании строки с использованием string в C#, 
символ новой строки \n автоматически добавляется в конце строки.

9. Каковы преимущества и недостатки использования операторов верхнего уровня при создании программ без метода Main?
Преимущества
Меньше шаблонного кода без явного метода Main
Более читаемый и компактный код

Недостатки:
Могут быть проблемы с порядком выполнения кода в некоторых случаях
Не подходят для случаев требующих дополнительной логики и структуры кода в методе Main
