2.   Какова каково назначение констант в C#? Как объявляется константная переменная и каковы ее характеристики?
- Константы предназначены для хранения значений которые не должны изменяться в течение выполнения программы.

const int MaxValue = 100;

Характеристики:
Значение константы должно быть установлено на этапе компиляции.
Используется для представления фиксированных значений.

4.   Объясните использование реляционных операторов в C# и как они определяют связь между операндами.
- Используются для сравнения значений и определения отношений между операндами.

int a = 5;
int b = 10;
bool isEqual = (a == b); // false
bool isNotEqual = (a != b); // true 

5.   Чем унарные операторы отличаются от других операторов, 
и каковы некоторые примеры унарных операторов в C#? унарных операторов в C#?
- Операторы применяемые к одному операнду.

int x = 5;
int y = -x; // Унарный минус
bool isTrue = true;
bool isFalse = !isTrue; // Логическое отрицание

Унарные операторы могут выполнять различные действия в зависимости от их типа
такие как изменение знака числа или инверсия логического значения.
